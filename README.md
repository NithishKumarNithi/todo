<h1>Fullstack TodoList Application</h1>

<p>Appliction contains two module client and server.</p>

<h2>Client Module</h2>
<p>Client Module contain code related to frontend of the application</p>
<p>Application Frontend is developed using React + Typescript with vite tool</p>

<h3>To run frontend</h3>
<p>Change directory to <em>client</em> and run <pre>npm install</pre></p>
<p>it will install necessary dependencies for react and typescript</p>
<p>Open new terminal in VSCode Editor and run <pre>npm run dev</pre></p>
<p>The frontend application will run on your http://localhost:5173</p>

<h2>Server Module</h2>
<p>Server Module contain code related to backend RestAPI of the application</p>
<p>Application Backend is developed using Nodejs and Express</p>


<h3>To run Backend</h3>
<p>Change directory to <em>Server</em> and run <pre>npm install</pre></p>
<p>it will install necessary dependencies for nodejs(>21.0.0) and express</p>
<p>Open seperate terminal in VSCode Editor and run <pre>npm run dev</pre></p>
<p>it will run the backend API service which provides resources to your frontend </p>
<p>The backend application will run on your http://localhost:7005</p>
